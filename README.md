# IBM_Exploration
This repository contains the initial exploration phase for developing an AI-powered virtual pet aimed at providing companionship and support for elderly users. The project leverages Unity for building an interactive environment and IBM Watson for natural language processing and voice interactions.


This project explores the development of an AI-powered virtual “social pet” designed to assist and engage elderly users. The virtual pet will provide companionship, encourage social interaction, and offer practical support through voice-based communication using IBM Watson services (Speech to Text, Text to Speech, and Watson Assistant). The system will be built in Unity, focusing on user-friendly design, emotional engagement, and real-time assistance.

Key features of the virtual pet include:
	•	Daily Routine Monitoring: Tracks the user’s routine and notifies caregivers if unusual activity is detected.
	•	Conversational Engagement: Prompts users to share their day and plan upcoming activities.
	•	Personalized Recommendations: Suggests music, podcasts, and other content based on user preferences.
	•	Messaging Assistance: Reads messages from family members and allows users to respond via voice.
	•	Emotional Interaction: Displays emotions (happy, sad, etc.) based on user interaction levels.

This project will initially focus on exploring tools, technologies, and frameworks necessary for development, including setting up Unity, integrating IBM Watson APIs, and designing basic virtual pet interactions.
